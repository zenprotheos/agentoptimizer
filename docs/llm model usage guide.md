### When to Use Which?

1. **Need ultra-cheap, snappy, multimodal & reasoning tasks?**
   → **GPT-5 Nano**.

2. **Want openness, privacy, local control, and strong reasoning/math?**
   → **OSS-120B via OpenRouter**.

3. **Working with enormous docs or long context reasoning?**

4. **Want a sweet spot of reasoning strength and cost—plus multimodal?**
